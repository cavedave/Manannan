# Agreed fixes for old-orthography transcription

This list records recurring corrections we agree to apply while proofreading old-orthography chapters against PDF scans.

- `scan header/footer text` -> `remove` (page artifacts like broken "manannán" banners are not chapter prose)
- `random internal capitals in normal words` -> `normalize to lowercase` (for readability when clearly OCR noise)
- `eitiLt` -> `eitilt` (mixed-case OCR noise)
- `eaglago` -> `eagla go` (missing word break)
- `tamailLín` -> `tamaillín` (mixed-case OCR noise)
- `an onn` -> `anonn` (split word OCR issue)
- `orÉa` -> `orṫa` (clear letter confusion in context)
- `arísaan` -> `arsa an` (joined words OCR issue)
- `mbóṫari` -> `mbóṫar` (spurious trailing character)
- `spacing before punctuation` -> `remove extra space` (e.g., `é ?` -> `é?`)
- `de’ii` -> `de’n` (common OCR confusion in contractions)
- `coṁ`/`ċom` in comparisons -> normalize to chapter style where context is clear
- `word-initial quote spacing` -> remove extra leading space after `“`
- `embedded full stops in words` -> remove when clearly OCR noise (e.g., `mar.do` -> `mar do`)
- `split or merged common words` -> normalize by context (e.g., `ṁui-.ntir` -> `ṁuintir`)
- `tÁrd-Ṁáiġistir` -> `tárd-Ṁáiġistir` (OCR uppercase accent issue in prefixed form)
- `stray hyphen at word end` -> `remove or merge` (OCR line-wrap noise, e.g. `ḃí-`, `coille-` before a space; confirm against PDF)
- `aGUs` / similar mixed-case `agus` -> `agus` (OCR noise)
- `Ḋo` (dotted capital) at start of `Do ṫóg` / particle `do` + lenited verb -> `Do` (plain capital D; lenition applies to the verb, not the particle)
- `. agus` at true sentence boundary in dialogue -> `. Agus` (capital after full stop)
- `go leir` (missing accent on *léir*) -> `go léir` when context is “all / entirely”
- `gCiandracán` after `an` (eclipsis) -> `gciandracán` (lowercase *c*)
- `rt-alṫóir` / similar hyphen-split `réaltóir` -> `réaltóir`
- `realtóir` (missing acute on *é*) -> `réaltóir` when meaning “astronomer”
- `agns` -> `agus` (OCR)
- `díriġṫr` -> `díriġṫe` when meaning “directed”
- `Ta` / `ta` (missing acute) before predicate -> `Tá` / `tá` (copula)
- `d aon` in `d’aon ghuth` / `d’aon` -> `d’aon` (one apostrophe)
- `na baineann` / `se` / `doman` in “you know (that) it only takes a year…” contexts -> often `go mbaineann sé` … `doṁan` (confirm verb against PDF: `dtógann` etc.)
- `dteasbáint` -> `dtaisebáint` (verbal noun “showing/displaying”; OCR letter confusion)
- `arsaan` -> `arsa an` (missing space in `arsa an`)
- `Dob ḟíor` -> `D’ḃí ḟíor` (“it was true”)
- `Rá` at sentence start before noun (copula sense) -> `Tá` when meaning “is”
- `Ḋeineas` (context: “I examined…”) -> `D’ḟéineas` (confirm against PDF; alternative forms may exist)
